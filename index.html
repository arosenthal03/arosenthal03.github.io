TCP
Transmisstion Control Protocol is a method (protocol) used along with Internet Protocol (IP) to send data in the form of messege units, called packets, between computers over the Internet.
IP
Internet Protocol is the method or protocol by which data is sent from one computer to another on the internet. Each computer on the Internet is uniquely identified by an IP adress.
FTP
File Transfer Protocol is a protocol used to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocol is the protocol used for exchanging text, graphic images, sound, video, and other multimedia files on the Web
